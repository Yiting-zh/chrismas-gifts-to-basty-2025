# 🎄 圣诞礼物网页 - Christmas Gifts for Basty 2025

一个温馨、欢快、充满北欧圣诞氛围的互动礼物网页，专为 Basty 准备的 2025 年圣诞礼物！

## ✨ 功能特点

- 🎅 **欢迎页面**：精美的圣诞树动画和装饰，营造温馨的圣诞氛围
- 🎁 **互动礼物**：4 个可点击的礼物盒，每个都有打开前后的不同状态
- ❄️ **飘雪效果**：持续的雪花飘落动画，增添节日气氛
- 🎨 **精美动效**：流畅的页面切换、礼物打开动画和悬停效果
- 📱 **响应式设计**：完美适配各种设备屏幕

## 🎯 交互流程

1. **欢迎页面** → 显示圣诞树、礼物堆和祝福语
2. **选择拆礼物方式**：
   - 点击 "现在拆礼物" → 进入礼物页面
   - 点击 "保留惊喜" → 显示倒计时（13天）
3. **礼物页面** → 4 个礼物盒可以逐个点击打开
4. **打开礼物** → 显示产品图片和描述
5. **完成页面** → 打开所有礼物后自动显示完成祝福

## 🚀 使用方法

### 1. 准备产品图片

将你准备的 4 张产品图片放入 `assets` 文件夹，并命名为：
- `gift1.jpg` - 礼物 1
- `gift2.jpg` - 礼物 2
- `gift3.jpg` - 礼物 3
- `gift4.jpg` - 礼物 4

> 💡 **提示**：支持 JPG、PNG 等格式，建议图片尺寸为正方形（如 500x500px）以获得最佳显示效果。

### 2. 打开网页

有以下几种方式打开网页：

#### 方式 A：直接双击打开
- 在文件管理器中找到 `index.html`
- 双击文件，会在默认浏览器中打开

#### 方式 B：使用 VS Code 的 Live Server（推荐）
1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"
4. 网页会在浏览器中自动打开并支持热重载

#### 方式 C：使用 Python 本地服务器
```bash
# 在项目目录下运行
python -m http.server 8000
```
然后在浏览器中访问 `http://localhost:8000`

### 3. 自定义内容

你可以轻松自定义以下内容：

#### 修改祝福语（index.html）
找到以下代码并修改：
```html
<h1 class="greeting-title">🎅 圣诞快乐！Basty 🎅</h1>
<p class="greeting-text">
    亲爱的 Basty，<br>
    在这个充满温暖与欢乐的圣诞节，<br>
    我为你准备了 4 份特别的礼物！✨<br>
    你想要现在就拆开它们，还是保留惊喜呢？
</p>
```

#### 修改礼物描述（index.html）
找到 `.gift-description` 类的段落：
```html
<p class="gift-description">你的第一份礼物！</p>
```
改为更具体的描述，如：
```html
<p class="gift-description">🎧 无线蓝牙耳机</p>
```

#### 修改倒计时天数（index.html）
找到倒计时数字并修改：
```html
<span class="countdown-number">13</span>
```

#### 调整颜色主题（styles.css）
主要颜色变量位于各个元素的样式中，可以搜索以下颜色值并替换：
- `#c41e3a` - 圣诞红
- `#2c5f2d` - 圣诞绿
- `#d4af37` - 金色
- `#667eea` - 背景渐变色

## 📝 注意事项

1. **图片文件**：请确保将 4 张产品图片放入 `assets` 文件夹，否则会显示图片加载失败
2. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
3. **文件路径**：如果移动文件位置，请确保相对路径正确
4. **视频支持**：如果想添加视频，可以将 `<img>` 标签替换为 `<video>` 标签

## 🎁 添加视频支持（可选）

如果想要某个礼物显示视频而不是图片，可以这样修改：

在 `index.html` 中找到对应礼物的 `.gift-content`，将：
```html
<img src="assets/gift1.jpg" alt="礼物 1" class="gift-image">
```

替换为：
```html
<video class="gift-image" controls autoplay loop>
    <source src="assets/gift1.mp4" type="video/mp4">
    您的浏览器不支持视频播放。
</video>
```

## 🎵 添加音效（可选）

如果想添加音效，在 `assets` 文件夹中放入音频文件，然后在 `script.js` 中取消音效相关代码的注释。
