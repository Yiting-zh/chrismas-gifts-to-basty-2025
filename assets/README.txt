# 占位图片 - 请替换为实际的产品图片

此文件夹用于存放礼物的产品图片。

## 需要添加的图片：

1. gift1.jpg - 礼物 1 的产品图片
2. gift2.jpg - 礼物 2 的产品图片
3. gift3.jpg - 礼物 3 的产品图片
4. gift4.jpg - 礼物 4 的产品图片

## 图片要求：

- 格式：JPG、PNG 或 GIF
- 建议尺寸：500x500px（正方形）
- 文件大小：建议小于 2MB 以确保快速加载

## 如何添加图片：

1. 将你的产品图片重命名为 gift1.jpg、gift2.jpg、gift3.jpg、gift4.jpg
2. 将这些图片复制到当前文件夹（assets）
3. 刷新网页即可看到效果

如果想使用其他格式（如 PNG），请相应修改 index.html 中的文件扩展名。
